# ⛰️ Crest: Product Status & Roadmap

This document provides a transparent overview of what Crest can do today and where it is headed. Our goal is to build the most powerful, private productivity tool for the Linux desktop.

## 🚀 Current Capabilities (v0.2.0)

### 📋 Clipboard Management
- **Native Listener**: Background thread monitors clipboard changes with zero CPU overhead.
- **SQLite Storage**: All copies are stored in a secure local database for instant retrieval.
- **Fuzzy Search**: Filter through thousands of past clips in milliseconds.

### 🌐 Intelligence & Search
- **Instant Web Answers**: Integrated DuckDuckGo bridge for Wikipedia summaries and quick facts.
- **Rich Result Previews**: Multi-line snippets for web results, including descriptions and site icons.
- **URL Navigation**: Direct jump to websites from the search bar (e.g., typing `iamk.xyz` opens your browser).

### 🧮 Utility Tools
- **Built-in Calculator**: Evaluate complex math expressions inline.
- **App Launcher**: Instant fuzzy-matching for all installed `.desktop` applications.
- **File Indexing**: Fast local file search with directory-aware scoring.

### 💎 UX & UI
- **Raycast Parity**: 100% glassmorphism design with ⌘K Action Bar support.
- **Keyboard-First**: Optimized for no-mouse workflows (Arrows, Enter, Backspace-to-mode).
- **Sub-50ms Response**: Built in Rust for "feels-like-native" latency.

## 🗺️ Upcoming Features (Roadmap)

### 🧩 Plugin Marketplace (Q3 2024)
- **Central Registry**: Browse and install community-built extensions with one click.
- **SDK Hardening**: More robust API for JS and Rust plugin developers.

### 🤖 Local AI Assistant (Q4 2024)
- **Ollama Integration**: Run local LLMs directly inside Crest for context-aware assistance.
- **Zero-Cloud AI**: Get AI help without your data ever leaving your machine.

### 🛠️ System Control Center
- **Core Toggles**: Manage Bluetooth, Wi-Fi, Dark Mode, and Volume directly from the launcher.
- **Window Management**: Quick-actions to tile, hide, or switch between active windows.

### 📱 Remote Bridge
- **Encrypted Local Remote**: Use your smartphone as a remote control for your Linux workstation over the local network.
