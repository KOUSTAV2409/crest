File Search

Privacy Promise

We collect absolutely nothing.

This is not a privacy policy buried in legalese. This is a table you can actually read.

What Crest (you) Cloud Launchers
Your search queries ❌ Never stored ⚠️ Often logged for analytics
Apps you launch ❌ Never tracked ⚠️ Usage data sent to servers
File names ❌ Stays on disk ⚠️ May be indexed in cloud
Clipboard contents ❌ Local SQLite only ⚠️ Cloud sync enabled by default
Internet connection required ✅ Optional (currency only) ⚠️ Required for most features
Source code available ✅ 100% open source (MIT) ❌ Closed source
Cost ✅ Free forever 💰 $8–16/month for Pro features

The currency exchange rate fetcher is the only optional network request Crest makes — and it fetches from a public API with no authentication, no user ID, and no identifying information. You can verify this by reading the source code directly.

Installation

Up and running in 60 seconds.

1

Prerequisites

Make sure you have Rust and Node.js installed.

bash 
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
# Then install Node.js via your package manager
sudo apt install nodejs npm   # Ubuntu/Debian
sudo pacman -S nodejs npm     # Arch Linux
2

Clone & Install

bash 
git clone https://github.com/KOUSTAV2409/crest.git
cd crest
npm install
3

Run

bash 
npm run tauri dev

Press Ctrl + Space to open the launcher from anywhere.
