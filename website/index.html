Technical Proof

Zero latency by design.

Crest is written in pure Rust, interfacing directly with X11/Wayland primitives. We don't use Electron. We don't use high-level abstractions that introduce garbage collection pauses.

bolt

Sub-1ms IPC

Custom binary protocol between UI and core engine.

memory

Low Memory Footprint

< 40MB RAM usage in idle state.
