# Crest v0.2.6 — The AppStream Integration

This release refines Crest's professional presence on Linux by integrating AppStream metadata, ensuring it looks stunning in software centers like GNOME Software and Discover.

### 🏔️ Highlights:
- **Professional Store Presence**: Full integration of `metainfo.xml` with high-fidelity screenshots, branding colors, and detailed release history.
- **Official DEB Bundling**: The metadata is now officially packaged into the `.deb` installer for seamless integration with system app stores.
- **Visual Revolution (v0.2.5 Core)**: Includes the ground-up UI overhaul, 50/50 split layout, and glassy command palette aesthetics.

### 🛠️ Changes:
- Added `dev.koustav.crest.metainfo.xml`.
- Updated `tauri.conf.json` with Linux bundling paths.
- Bumped project-wide versioning for a clean CI build.
